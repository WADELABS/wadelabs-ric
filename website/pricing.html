COMMON QUESTIONS

DO YOU NEED SOURCE CODE ACCESS?

No. RIC operates entirely from the outside, just like a user or attacker. We only need staging credentials.

HOW IS THIS DIFFERENT FROM QA?

QA tests features ("Does the button work?"). RIC tests guarantees ("Does the button bypass auth?"). We assume the feature works and test if it breaks integrity.

CAN I CANCEL ANYTIME?

Yes. Contracts are monthly. Integrity is continuous, but our agreement doesn't have to be locked in.
